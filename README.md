# 🧠 Repositório de Imagens — Inteligência Artificial (UFPA)

Este repositório foi criado com o objetivo de armazenar e organizar imagens utilizadas em notebooks Jupyter ao longo da disciplina de Inteligência Artificial, no curso de Ciência da Computação da Universidade Federal do Pará (UFPA).

As imagens aqui hospedadas serão utilizadas em um repositório principal, servindo como apoio para:

* apresentações
* experimentos e análises

---
## 🔗 Utilização

As imagens deste repositório podem ser acessadas diretamente via link e utilizadas em arquivos Markdown no repositório principal. Exemplo:

```markdown
![Descrição da imagem](https://raw.githubusercontent.com/Itallo0708/IA-imagens/main/caminho/da/imagem.png)
```

---

## ⚠️ Observações

* Este repositório não contém código executável.
* O foco é exclusivamente o armazenamento de **recursos visuais**.
